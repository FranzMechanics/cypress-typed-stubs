# Cypress typed stubs

[![License](https://img.shields.io/badge/license-apache-blue.svg)](LICENSE)

> Tools to define **typed** Cypress stubs based on clients generated with Swagger.

Automatically built from clients:
- HTTP method
- URL pattern
- name
